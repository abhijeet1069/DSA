A collection of DSA ideas which are quite counter-intutive, but represent the beauty of the subject.

DATA-STRUCTURES:

ARRAYS
1) ds.arrays.ex02.CheckMergeArray - Merging array using only while loops

2) ds.arrays.ex04.ProductArray - This type of breaking a problem into subproblems is quite 
        an effective way of solving problems. Here, the product to be found is broken into left and right component.

3) ds.arrays.ex07.CheckSecondMax - 'if' of max and 'if' of sec_max are complementing each other in a wierd way.

4) ds.arrays.ex11.FindMax - Use Kadane algorithm

LINKED-LIST
5) ds.linkedList.ex07.CheckMid - Using fast and slow pointer to find the mid of linkedlist.

STACKS/QUEUES:
6) ds.stackQueue.ex01.FindBinChallenge - Finding binary numbers from 1 to n. Previous element + 0 and previous element+ 1 . base case = 1
        Using queue for recursive definition.

7) ds.stackQueue.ex06.EvaluatePostfixChallenge - Push the operands while pop the operands when an operator is encountered. Here, we have binary operator
        so we pop two elements.

8) ds.stackQueue.ex06.NextGreaterChallenge - Using stack to keep a record of recent high. And starting from last.

9) ds.stackQueue.ex09.CheckBalancedChallenge - Using stack to check if an expression is balanced or not.

GRAPHS:



10) ds.graphs.ex01.CheckBFS - BFS is same as DFS except that DFS uses stack instead of queue

11) ds.graphs.ex07.CheckTree - A graph is a tree if there is connected, has no cycle and each node has only one parent

12) ds.graphs.ex08.CheckMin -  Main logic is to find vertices level using BFS. And then find min level of connected node to destination + 1
