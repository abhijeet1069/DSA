A binary heap is a complete binary tree which satisfies the heap ordering property.

Complete Binary Tree : 
	A complete binary tree is a tree where each node has a max of 2 children and nodes at all levels are completely filled(except the leaf nodes). 
	But the nodes at the last level must be structured in such a way that the left side is never empty.

Heap Property
	Min Heap
		All the parent node keys are less than or equal to their child node keys.
	Max Heap
		All the parent node keys must be greater than or equal to their child node keys.
		
The major uses of heap are:
	a) Order statistics
	b) Priority Queues
	c) Sorting
